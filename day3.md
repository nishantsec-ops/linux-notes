## Commands Cheat Sheet

- `ls` тЖТ List files and directories  
- `pwd` тЖТ Show current working directory  

- `cat` тЖТ Display file contents  
- `head` тЖТ Show first 10 lines  
- `head -n 1` тЖТ Show first line  
- `tail` тЖТ Show last 10 lines  
- `tail -n 2` тЖТ Show last 2 lines  
- `less` тЖТ View file page by page  

- `wc` тЖТ Count lines, words, characters  
- `wc -l` тЖТ Count lines  
- `wc -w` тЖТ Count words  
- `wc -c` тЖТ Count characters  

- `history` тЖТ Show command history  
- `!number` тЖТ Run command from history
